408 /************************************************************************************
409  * Name: uart_connected
410  *
411  * Description:
412  * Serial devices (like USB serial) can be removed. In that case, the "upper
413  * half" serial driver must be informed that there is no longer a valid serial
414  * channel associated with the driver.
415  *
416  * In this case, the driver will terminate all pending transfers wint ENOTCONN and
417  * will refuse all further transactions while the "lower half" is disconnected.
418  * The driver will continue to be registered, but will be in an unusable state.
419  *
420  * Conversely, the "upper half" serial driver needs to know when the serial
421  * device is reconnected so that it can resume normal operations.
422  *
423  * Assumptions/Limitations:
424  * This function may be called from an interrupt handler.
