60 /* Memory Map
61  *
62  * 64Kb Program Memory (64K series)
63  * C:0000 - C:0001 : Flash options
64  * C:0002 - C:0037 : Vectors
65  * : ROM data
66  * : Code
67  *
68  * 4Kb Register File (64K series)
69  * R:020 - R:0ff : 224 byte RDATA
70  * R:0e0 - R:0ef : 16-byte working register area (RDATA)
71  * E:100 - E:eff : 3.5 Kbyte EDATA
72  * f00 - fff : 256 byte control register area
73  */
