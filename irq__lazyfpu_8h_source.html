75 /* If the MCU supports a floating point unit, then it will be necessary
76  * to save the state of the FPU status register and data registers on
77  * each context switch. These registers are not saved during interrupt
78  * level processing, however. So, as a consequence, floating point
79  * operations may NOT be performed in interrupt handlers.
80  *
81  * The FPU provides an extension register file containing 32 single-
82  * precision registers. These can be viewed as:
83  *
84  * - Sixteen 64-bit doubleword registers, D0-D15
85  * - Thirty-two 32-bit single-word registers, S0-S31
86  * S<2n> maps to the least significant half of D<n>
87  * S<2n+1> maps to the most significant half of D<n>.
88  */
89
