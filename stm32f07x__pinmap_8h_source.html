51 /* Alternate Pin Functions.
52  *
53  * Alternative pin selections are provided with a numeric suffix like _1, _2, etc.
54  * Drivers, however, will use the pin selection without the numeric suffix.
55  * Additional definitions are required in the board.h file. For example, if
56  * CAN1_RX connects vis PD0 on some board, then the following definition should
57  * appear inthe board.h header file for that board:
58  *
59  * #define GPIO_CAN1_RX GPIO_CAN1_RX_1
60  *
61  * The driver will then automatically configre PD0 as the CAN1 RX pin.
62  */
63
