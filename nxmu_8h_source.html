477 /****************************************************************************
478  * Name: nxfe_constructwindow
479  *
480  * Description:
481  * This function is the same a nx_openwindow EXCEPT that the client provides
482  * the window structure instance. nx_constructwindow will initialize the
483  * the pre-allocated window structure for use by NX. This function is
484  * provided in addition to nx_open window in order to support a kind of
485  * inheritance: The caller's window structure may include extensions that
486  * are not visible to NX.
487  *
488  * NOTE: wnd must have been allocated using kmm_malloc() (or related allocators)
489  * Once provided to nxfe_constructwindow() that memory is owned and managed
490  * by NX. On certain error conditions or when the window is closed, NX will
491  * free the window.
492  *
493  * Input Parameters:
494  * handle - The handle returned by nx_connect
495  * wnd - The pre-allocated window structure.
496  * cb - Callbacks used to process window events
497  * arg - User provided value that will be returned with NX callbacks.
498  *
499  * Return:
500  * OK on success; ERROR on failure with errno set appropriately. In the
501  * case of ERROR, NX will have deallocated the pre-allocated window.
502  *
503  ****************************************************************************/
504 
505 int nxfe_constructwindow(NXHANDLE handle,
506  FAR struct nxbe_window_s *wnd,
507  FAR const struct nx_callback_s *cb,
508  FAR void *arg);
