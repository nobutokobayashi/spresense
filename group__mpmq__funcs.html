int mpmq_init ( mpmq_t *  mq,
key_t  key,
cpuid_t  cpu 
)

Initialize MP message queue

mpmq_init() initialize MP message queue object. This function needs raw CPU ID at cpuid. User can use mptask_getcpuid() to get asigned CPU ID. When call this API on the worker side, cpuid is ignored.

Parameters
[in,out]mqMP message queue object
[in]keyUnique object ID
[in]cpuidTarget CPU ID
Returns
On success, mpmq_init() returns 0. On error, it returns an error number.
Return values
-EINVALInvalid argument
