73  * produce a 2MHz reference clock to the PLL. The rated speed for the MK20DX256VLH7
74  * is 72MHz and 50MHz for the MK20DX128VLH5, but according to the PJRC website,
75  * both can be overclocked at 96MHz
76  *
77  * MK20DX128VLH5 Rated Frequency 50MHz (selecting 48Mhz to use USB)
78  *
79  * PLL Input frequency: PLLIN = REFCLK/PRDIV = 16MHz/8 = 2MHz
80  * PLL Output frequency: PLLOUT = PLLIN*VDIV = 2Mhz*24 = 48MHz
81  * MCG Frequency: PLLOUT = 48MHz
82  *
83  * MK20DX256VLH7 Rated Frequency 72MHz
84  *
85  * PLL Input frequency: PLLIN = REFCLK/PRDIV = 16MHz/8 = 2MHz
86  * PLL Output frequency: PLLOUT = PLLIN*VDIV = 2Mhz*36 = 72MHz
87  * MCG Frequency: PLLOUT = 72MHz
88  *
89  * Board can be overclocked at 96MHz (per PJRC.com)
90  * PLL Input frequency: PLLIN = REFCLK/PRDIV = 16MHz/8 = 2MHz
91  * PLL Output frequency: PLLOUT = PLLIN*VDIV = Mhz*48 = 96MHz
