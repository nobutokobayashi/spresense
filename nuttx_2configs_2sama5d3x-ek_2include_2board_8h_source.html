123 /* LED definitions ******************************************************************/
124 /* There are two LEDs on the SAMA5D3 series-CM board that can be controlled
125  * by software. A blue LED is controlled via PIO pins. A red LED normally
126  * provides an indication that power is supplied to the board but can also
127  * be controlled via software.
128  *
129  * PE25. This blue LED is pulled high and is illuminated by pulling PE25
130  * low.
131  *
132  * PE24. The red LED is also pulled high but is driven by a transistor so
133  * that it is illuminated when power is applied even if PE24 is not
134  * configured as an output. If PE24 is configured as an output, then the
135  * LCD is illuminated by a high output.
136  */

175  * the red LED that blinks in the event of an PANIC.

177  * Thus if the blue LED is statically on, NuttX has successfully booted and

183 /* Button definitions ***************************************************************/
184 /* There are five push button switches on the SAMA5D3X-EK base board:
185  *
186  * 1. One Reset, board reset (BP1)
187  * 2. One Wake up, push button to bring the processor out of low power mode
188  * (BP2)
189  * 3. One User momentary Push Button
190  * 4. One Disable CS Push Button
191  *
192  * Only the momentary push button is controllable by software (labelled
193  * "PB_USER1" on the board):
194  *
195  * - PE27. Pressing the switch connect PE27 to grounded. Therefore, PE27
196  * must be pulled high internally. When the button is pressed the SAMA5
197  * will sense "0" is on PE27.
198  */
199
