118 /* Register bit definitions *********************************************************/
119 /* There are no bit field definitions for the following registers because they support
120  * 32-bit values:
121  *
122  * - Timer Counter register, channel 0 (TC0), Timer Counter register, channel 1 (TC1),
123  * and Timer Counter register, channel 2 (TC2): 32-bit Timer/Counter values for
124  * channels 0, 1, 2 (no bit field definitions)
125  *
126  * - Limit register, channel 0 (LIM0), Limit register, channel 1 (LIM1), and Limit
127  * register, channel 2 (LIM2): 32-bit Limit values for TC0, 1, 2 (no bit field
128  * definitions)
129  *
130  * - Match register, channel 0 MAT0), Match register, channel 1 (MAT1), and Match
131  * register, channel 2 (MAT2): 32-bit Match values for TC0, 1, 2 (no bit field
132  * definitions).
133  *
134  * - Capture register, channel 0 (CAP0), Capture register, channel 1 (CAP1), and
135  * Capture register, channel 2 (CAP2): 32-bit TC value at a capture event for
136  * channels 0, 1, 2 (no bit field definitions)
137  */
