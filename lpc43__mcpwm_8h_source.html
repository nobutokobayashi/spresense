115 /* Register bit definitions *********************************************************/
116 /* There are no bit field definitions for the following registers because they support
117  * 32-bit values:
118  *
119  * - Timer Counter register, channel 0 (TC0), Timer Counter register, channel 1 (TC1),
120  * and Timer Counter register, channel 2 (TC2): 32-bit Timer/Counter values for
121  * channels 0, 1, 2 (no bit field definitions)
122  *
123  * - Limit register, channel 0 (LIM0), Limit register, channel 1 (LIM1), and Limit
124  * register, channel 2 (LIM2): 32-bit Limit values for TC0, 1, 2 (no bit field
125  * definitions)
126  *
127  * - Match register, channel 0 MAT0), Match register, channel 1 (MAT1), and Match
128  * register, channel 2 (MAT2): 32-bit Match values for TC0, 1, 2 (no bit field
129  * definitions).
130  *
131  * - Capture register, channel 0 (CAP0), Capture register, channel 1 (CAP1), and
132  * Capture register, channel 2 (CAP2): 32-bit TC value at a capture event for
133  * channels 0, 1, 2 (no bit field definitions)
134  */
