51 /* Flash size is known from the chip selection:
52  *
53  * When CONFIG_STM32L4_FLASH_OVERRIDE_DEFAULT is set the
54  * CONFIG_STM32L4_FLASH_CONFIG_x selects the default FLASH size based on the chip
55  * part number. This value can be overridden with CONFIG_STM32L4_FLASH_OVERRIDE_x
56  *
57  * Parts STM32L4xxE have 512Kb of FLASH
58  * Parts STM32L4xxG have 1024Kb of FLASH
59  *
60  * N.B. Only Single bank mode is supported
61  */
