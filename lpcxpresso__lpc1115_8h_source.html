156 /* USB:
157  *
158  * Base-board J4/J6 LPC1115
159  * Signal Pin Pin
160  * ------------------- ----- --------
161  * USB_DM 36 P0.30/USB-D-
162  * USB_DP 37 P0.29/USB-D+
163  * N/C N/C P1.18/USB-UP-LED/PWM1.1/CAP1.0
164  * PIO1_3 (not used) 19 P1.30/VBUS/AD0.4
165  * N/C N/C P2.9/USB-CONNECT/RXD2 (See Notes)
166  * ------------------- ----- --------
167  * PIO0_6-USB_CONNECT* 23 P0.21/RI1/RD1
168  * PIO0_3-VBUS_SENSE 39 P0.5/I2SRX-WS/TD2/CAP2.1
169  *
170  * Notes:
171  * - The standard USB CONNECT (P0.9) provides USB D+ pullup on board the
172  * LPCXpresso card; it should be un-necessary to use the based board
173  * version of the pullup.
174  * - No changes to jumper settings are required. There are few USB-
175  * related jumpers on the based board, but none are required:
176  * - J14 must be set to permit GPIO control of the base board USB
177  * connect pin. NOT USED
178  * - J12 must be set to permit GPIO control of the USB vbus sense pin
179  * - The standard VBUS (P1.30) is not connected. As a consequence, the
180  * USB driver will not correctly respond to USB insertion or removal
181  * events.
182  * - The standard USB LED (P1.18) is not connected.
183  */
