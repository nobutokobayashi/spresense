Detailed Description

signal setting for reading data asychronously The field 'enable' of cxd56_gnss_signal_setting to be given as a parameter must be specified as 1 when setting and 0 when unsetting. Field 'gnsssig' specifies the value of 'Signal types from GNSS', this is not POSIX signal. Field 'signo' is application specific number of POSIX signal. 'data' will be passed as an argument to the handler.
