233 /************************************************************************************
234  * Name: sam_ohci_initialize
235  *
236  * Description:
237  * Initialize USB OHCI host controller hardware.
238  *
239  * Input Parameters:
240  * controller -- If the device supports more than one USB OHCI interface, then
241  * this identifies which controller is being initializeed. Normally, this
242  * is just zero.
243  *
244  * Returned Value:
245  * And instance of the USB host interface. The controlling task should
246  * use this interface to (1) call the wait() method to wait for a device
247  * to be connected, and (2) call the enumerate() method to bind the device
248  * to a class driver.
249  *
250  * Assumptions:
251  * - This function should called in the initialization sequence in order
252  * to initialize the USB device functionality.
253  * - Class drivers should be initialized prior to calling this function.
254  * Otherwise, there is a race condition if the device is already connected.
255  *
256  ************************************************************************************/
257
