57 /* For the TM4C129X family, the Flash memory is configured in groups of four banks
58  * four banks of 16K x 128 bits (4 * 256 KB total) which are two-way interleaved.
59  * Because the memory is two-way interleaved and each bank individually is an 8-KB
60  * sector, when the user erases a sector, using the ERASE bits in the Flash Memory
61  * Control (FMC) register, it is a 16 KB erase.
62  */
