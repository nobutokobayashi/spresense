211  /* Interface to control the ILI9340 lcd driver

213  * - select Select the device (as neccessary) before performing any operations.

215  * - sendcmd Send specific command to the LCD driver.
216  * - sendparam Send specific parameter to the LCD driver.
217  * - recvparam Receive specific parameter from the LCD driver.
218  * - sendgram Send pixel data to the LCD drivers gram.
219  * - recvgram Receive pixel data from the LCD drivers gram.
220  * - backlight Change the backlight level of the connected display.
221  * In the context of the ili9340 that means change the
222  * backlight level of the connected LED driver.
223  * The implementation in detail is part of the platform
224  * specific sub driver.
225  *
226  */

259 /**************************************************************************************
260  * Name: ili9340_initialize
261  *
262  * Description:
263  * Initialize the LCD video driver internal sturcture. Also initialize the
264  * lcd hardware if not done. The control of the LCD driver is depend on the
265  * selected MCU interface and part of the platform specific subdriver (see

270  * lcd - A reference to the platform specific driver instance to control the
271  * ili9340 display driver.
272  * devno - A value in the range of 0 through CONFIG_ILI9340_NINTERFACES-1.
273  * This allows support for multiple LCD devices.
274  *

284 /**************************************************************************************
285  * Name: ili9340_clear
286  *
287  * Description:
288  * This is a non-standard LCD interface. Because of the various rotations, clearing
289  * the display in the normal way by writing a sequences of runs that covers the
290  * entire display can be very slow. Here the display is cleared by simply setting
291  * all GRAM memory to the specified color.
292  *
293  * Parameter:
294  * dev - A reference to the lcd driver structure
295  * color - The background color
296  *
297  * Returned Value:
298  *
299  * On success - OK
300  * On error - -EINVAL
301  *
302  **************************************************************************************/
303 
304 int ili9340_clear(FAR struct lcd_dev_s *dev, uint16_t color);
