57 /* LEDs: There are three LEDs on board the SAML21 Xplained Pro board: The EDBG
58  * controls two of the LEDs, a power LED and a status LED. There is only
59  * one user controllable LED, a yellow LED labelled STATIS near the SAML21 USB
60  * connector.
61  *
62  * This LED is controlled by PB10 and the LED can be activated by driving PB10
63  * to GND.
64  *
65  * When CONFIG_ARCH_LEDS is defined in the NuttX configuration, NuttX will
66  * control the LED as follows:
67  *
68  * SYMBOL Meaning LED
69  * ------------------- ----------------------- ------
70  * LED_STARTED NuttX has been started OFF
71  * LED_HEAPALLOCATE Heap has been allocated OFF
72  * LED_IRQSENABLED Interrupts enabled OFF
73  * LED_STACKCREATED Idle stack created ON
74  * LED_INIRQ In an interrupt N/C
75  * LED_SIGNAL In a signal handler N/C
76  * LED_ASSERTION An assertion failed N/C
77  * LED_PANIC The system has crashed FLASH
78  *
79  * Thus if the LED is statically on, NuttX has successfully booted and is,
80  * apparently, running normally. If the LED is flashing at approximately
81  * 2Hz, then a fatal error has been detected and the system has halted.
82  */
83 
84 #define PORT_STATUS_LED (PORT_OUTPUT | PORT_PULL_NONE | PORT_OUTPUT_SET | \
85  PORTB | PORT_PIN10)
