66 /* IOCTL Commands ***********************************************************/
67 /* The upper-half battery fuel gauge driver provides a character driver
68  * "wrapper" * around the lower-half battery driver that does all of the
69  * real work.
70  * Since there is no real data transfer to/or from a battery, all of the
71  * driver interaction is through IOCTIL commands. The IOCTL commands
72  * supported by the upper-half driver simply provide calls into the
73  * lower half as summarized below:
74  *
75  * BATIOC_STATE - Return the current state of the battery (see
76  * enum battery_gauge_status_e).
77  * Input value: A pointer to type int.
78  * BATIOC_ONLINE - Return 1 if the battery is online; 0 if offline.
79  * Input value: A pointer to type bool.
80  * BATIOC_VOLTAGE - Return the current battery voltage. The returned value
81  * is a fixed precision number in units of volts.
82  * Input value: A pointer to type b16_t.
83  * BATIOC_CAPACITY - Return the current battery capacity or State of Charge
84  * (SoC). The returned value is a fixed precision percentage of the
85  * batteries full capacity.
86  * Input value: A pointer to type b16_t.
87  */
88
