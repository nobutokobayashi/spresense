55 /* Command: GPIOC_WRITE
56  * Description: Set the value of an output GPIO
57  * Argument: T0=output a low value; 1=outut a high value
58  *
59  * Command: GPIOC_READ
60  * Description: Read the value of an input or output GPIO
61  * Argument: A pointer to an bool value to receive the result:
62  * false=low value; true=high value.
63  *
64  * Command: GPIOC_PINTYPE
65  * Description: Return the GPIO pin type.
66  * Argument: A pointer to an instance of type enum gpio_pintype_e
67  *
68  * Command: GPIOC_REGISTER
69  * Description: Register to receive a signal whenever there an interrupt
70  * is received on an input gpio pin. This feature, of course,
71  * depends upon interrupt GPIO support from the platform.
72  * Argument: The number of signal to be generated when the interrupt
73  * occurs.
74  *
75  * Command: GPIOC_UNREGISTER
76  * Description: Stop receiving signals for pin interrupts.
77  * Argument: None.
78  */
