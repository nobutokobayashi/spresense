void UM_Abort ( void  )

Abort update manager

This API freeing internal used memory and initialize update manager, but checkpoint is not cleared. User can be use this API to pause/resume or restart download sequence without heap dirty.

int UM_CheckPoint ( void  )

Create recovery check point.

User can use this feature to stop downloading (also shutdown). Pass true to need_recovery on UM_Init() to restart downloading, all of closed objects are scheduled again until this check point.

Check point is always overwritten.

Returns
0 on success, otherwise error
Return values
-EFAULTrecovery point not saved

Yes
