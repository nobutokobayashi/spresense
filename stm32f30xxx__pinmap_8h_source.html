53 /* Alternate Pin Functions. All members of the STM32F30xxx family share the same
54  * pin multiplexing (although they may differ in the pins physically available).
55  *
56  * Alternative pin selections are provided with a numeric suffix like _1, _2, etc.
57  * Drivers, however, will use the pin selection without the numeric suffix.
58  * Additional definitions are required in the board.h file. For example, if
59  * CAN1_RX connects vis PA11 on some board, then the following definitions should
60  * appear inthe board.h header file for that board:
61  *
62  * #define GPIO_CAN1_RX GPIO_CAN1_RX_1
63  *
64  * The driver will then automatically configre PA11 as the CAN1 RX pin.
65  */
